1. Layout Estático

    - El contenido tiene un ancho fijo (ej. width: 1200px) y no cambia con el tamaño de la pantalla.
    - No es flexible ni adaptable a diferentes dispositivos.
    - Se usaba comúnmente antes del diseño responsivo.

2. Layout Fluido (Liquid Layout)

    - Usa porcentajes en lugar de valores fijos (width: 80% en lugar de width: 1200px).
    - Se ajusta al ancho de la ventana del navegador, pero sin control total sobre el diseño.

3. Layout Responsivo

    - Utiliza Media Queries de CSS (@media) para adaptar el diseño a diferentes tamaños de pantalla.
    - Utiliza Media Queries de CSS (@media) para adaptar el diseño a diferentes tamaños de pantalla.
    - Es el estándar actual para el diseño web moderno.

4. Layout Adaptativo (Adaptive Layout)

    - Similar al responsivo, pero en lugar de cambiar gradualmente, usa tamaños predefinidos para distintos dispositivos.
    - Se basa en "breakpoints" específicos (ej. max-width: 768px).
    - Puede generar sitios más rápidos, pero menos flexibles.

5. Layout de una sola columna (Single-Column Layout)

    - Todo el contenido está en una sola columna, ideal para móviles.
    - Muy usado en blogs y artículos de lectura.

6. Layout de múltiples columnas (Multi-Column Layout)

    - Divide el contenido en varias columnas, útil para sitios con mucho contenido.
    - Se puede lograr con CSS Grid o Flexbox.s

7. Layout Basado en CSS Grid

    - Usa CSS Grid para crear estructuras de cuadrícula complejas y organizadas.
    - Permite alineaciones avanzadas y control total del diseño.

8. Layout Basado en Flexbox

    - Se usa para alinear elementos en una dimensión (fila o columna).
    - Ideal para menús, barras de navegación y alineaciones horizontales o verticales.
